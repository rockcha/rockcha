현재는 React, Next.js, TypeScript 중심의 웹 개발에 집중하고 있습니다. 아직 배워가는 단계이지만 유지보수하기 쉽고 구조가 명확한 애플리케이션을 만드는 것을 목표로 다양한 프로젝트를 제작하며 성장하고 있습니다.
