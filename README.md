## 🙋 저를 소개합니다

### 📘 도전과 학습

비전공자로서 `개발`이라는 새로운 도전은 처음엔 두려웠으나 제 자신의 학습능력을 믿었고 실제로 부딪혀가며 성장하는 것을 좋아하기 때문에, 게임 개발과 프론트엔드 공부에 도전할 수 있었습니다.

### 🌍 소통과 글로벌 역량

캐나다에서 1년간 어학연수를 했으며, 어릴 때부터 해외 영화를 시청하고 영어 서적을 읽는 것을 좋아했습니다. 또한 영어로 매일 일기를 작성하며 꾸준히 영어 역량을 키워왔습니다.

소통에도 강점이 있습니다. 20살 때부터 7~8년간 국어, 수학, 영어, 논술, 아랍어 등 다양한 과목으로 과외를 진행해왔으며, 대학교 재학 중에는 모의국무회의 학회의 회장을 맡아 약 1년간 집행부와 학회원들을 이끌었습니다. 매주 세미나를 통해 사회 문제를 주제로 연극을 기획, 준비했고 최종적으로 교수님과 학부모님들 앞에서 공연을 성공적으로 선보였습니다. 이 과정에서 기획부터 연습, 운영, 사회까지 전 과정을 주도적으로 수행했으며, 여러 이해관계자들과의 소통을 조율하는 경험을 할 수 있었습니다.

### 📹 지식 공유와 콘텐츠 활동

아는 것과 설명하는 것은 또 다른 영역이라고 생각합니다. 강의 형태의 유튜브 채널을 운영하며 학습한 내용을 정리하고 다른 사람에게 설명하는 연습을 하고 있습니다. 이 과정을 통해 이미 알고 있던 개념을 더 깊이 이해하고, 기술을 명확하게 전달하는 능력을 함께 키워가고 있습니다. 결과적으로 생각보다 많은 분들이 메일을 통해, 예제 코드를 요청하셨고 그럴 때마다 저 스스로가 사람들과 함께 성장하는 것을 좋아하는 사람임을 다시금 깨닫게 되었습니다.

- YouTube: [@codingfrog_kr](https://www.youtube.com/@codingfrog_kr)

### ✨ 프론트엔드로 확장한 계기

처음 개발을 접했을 때는 C++을 통한 알고리즘 공부로 시작하게 되었습니다. 백준과 프로그래머스 문제들을 해결해 나가며 개발의 재미에 매료되었고, 더 나아가 게임 개발에 도전하게 되었습니다. 그 과정에서 시스템적으로 사고하는 법과 프로그래밍에 친숙해지는 계기가 되었습니다.

그러던 중, 코멘토에서 주관하는 웹 접근성, 웹 표준 직무 체험에 참여하게 되었습니다. 이전까지는 프론트엔드 개발을 단순히 눈에 보이는 UI를 구현하는 역할로 생각했지만, 이 경험을 통해 소외되는 사람이 없이 모든 사용자가 동등하게 웹에 접근할 수 있도록 보이지 않는 부분까지 세심하게 설계하고 구현해야 한다는 점을 깨닫게 되었습니다. 실제로 웹 접근성과 표준을 고려한 기능들을 직접 구현해보며, 사용자 경험을 보다 넓은 관점에서 고민하는 프론트엔드 직무가 아름답다고 느꼈고 결과적으로, 프론트엔드의 진정한 가치와 매력에 깊이 빠지게 되었습니다.

### 🏆 프로젝트 성과와 현재 집중 분야

이후 한컴 AI 아카데미 2기를 수료하며, PM, LLM, 백엔드 담당 인원과 협업하여, 저는 프론트엔드를 맡아 AI 코딩 실력을 점수화하고 분석 결과를 기반으로 개선 코드를 제안하는 VS Code Extension ‘DKMV’를 개발하였습니다. GitHub OAuth를 통해 로그인하면, 익스텐션에서 수행된 AI 코드 분석 결과가 히스토리로 저장되며, 별도의 웹 서비스에서 해당 데이터를 기반으로 다양한 통계 정보를 제공합니다. 사용자는 월간 점수 추이, 유형별 점수, 사용자 간 순위 등 여러 지표를 통해 자신의 AI 코딩 역량을 지속적으로 점검하고 향상시킬 수 있습니다. 저는 익스텐션 개발과 웹 페이지 개발을 담당하였으며, 프론트엔드 구현에 그치지 않고 프로젝트의 전반적인 방향성과 데이터 처리 흐름 설계에도 적극적으로 참여하였습니다. 이를 통해 단순 UI 구현을 넘어, 서비스 구조를 이해하고 협업 속에서 문제를 해결하는 경험을 쌓을 수 있었습니다. 프로젝트 완성 이후에는 익스텐션을 마켓플레이스에 등록하고 실제로 시연을 진행하였으며, 한글과컴퓨터 본사 직원들 앞에서 최종 발표를 수행했습니다. 그 결과, 저희 팀은 최우수 팀으로 선정되었고, 본 프로젝트는 협업의 가치와 개발의 즐거움을 동시에 느낄 수 있었던 의미 있는 경험으로 남았습니다. 이 과정에서 단순히 AI를 사용하는 것을 넘어, 개발 과정에서 AI가 어떤 방식으로 가치를 만들어낼 수 있는지에 대해 깊이 고민하 게 되었고 기술을 도구로 활용하는 관점뿐 만 아니라 문제 해결의 새로운 접근 방식으로 바라 보는 계기를 얻게 되었습니다. 현재는 React, Next.js, TypeScript 중심의 웹 개발에 집중하고 있습니다. 아직 배워가는 단계이지만 유지보수하기 쉽고 구조가 명확한 애플리케이션을 만드는 것을 목표로 다양한 프로젝트를 제작하며 성장하고 있습니다.

### 🚀 앞으로의 포부
개발자라는 직업은 무한한 가능성을 지닌 분야라고 생각합니다. 모든 기능을 한 번에 완벽하게 구현할 수는 없지만, 필요한 내용을 스스로 학습하고 구현하며 지속적으로 발전해 나갈 수 있는 가능성이 있습니다. 물론 다른 직종 역시 성장의 기회가 존재하지만, 제가 느낀 개발자는 이러한 가능성이 특히 두드러지는 직업이었습니다. 이러한 개발자의 매력에 이끌려 포기하지 않고 여기까지 오게 되었습니다. 지금의 저는 아직 부족한 점이 많지만, 제 성장 가능성을 믿고 있습니다. 그리고 개발을 통해 끊임없이 배우고 성장하며, 그 가능성을 직접 증명해 나가고 싶습니다.
