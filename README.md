## 🙋 저를 소개합니다

### 📘 도전과 학습

비전공자로서 '개발'이라는 새로운 도전은 처음엔 두려웠으나 제 자신의 학습 능력을 믿었고 실제로 부딪혀가며 성장하는 것을 좋아하기 때문에, 게임 개발과 프론트엔드 공부를 해왔습니다.

### 🌍 소통과 글로벌 역량

캐나다에서 1년간 어학연수를 했으며, 어릴 때부터 해외 영화를 시청하고 영어 서적을 읽는 것을 좋아했습니다. 또한 영어로 매일 일기를 작성하며 꾸준히 영어 역량을 키워왔습니다.

소통에도 강점이 있습니다. 20살 때부터 7~8년간 국어, 수학, 영어, 논술, 아랍어 등 다양한 과목으로 과외를 진행해왔으며, 대학교 재학 중에는 모의국무회의 학회의 회장을 맡아 약 1년간 집행부와 학회원들을 이끌었습니다. 매주 세미나를 통해 사회 문제를 주제로 연극을 기획, 준비했고 최종적으로 교수님과 학부모님들 앞에서 공연을 성공적으로 선보였습니다. 이 과정에서 기획부터 연습, 운영, 사회까지 전 과정을 주도적으로 수행했으며, 여러 이해관계자들과의 소통을 조율하는 경험을 할 수 있었습니다.

### 📹 지식 공유와 콘텐츠 활동

아는 것과 설명하는 것은 또 다른 영역이라고 생각합니다. 강의 형태의 유튜브 채널을 운영하며 학습한 내용을 정리하고 다른 사람에게 설명하는 연습을 하고 있습니다. 이 과정을 통해 이미 알고 있던 개념을 더 깊이 이해하고, 기술을 명확하게 전달하는 능력을 함께 키워가고 있습니다. 결과적으로 생각보다 많은 분들이 메일을 통해, 예제 코드를 요청하셨고 그럴 때마다 저 스스로가 사람들과 함께 성장하는 것을 좋아하는 사람임을 다시금 깨닫게 되었습니다.

- YouTube: [@codingfrog_kr](https://www.youtube.com/@codingfrog_kr)

### ✨ 프론트엔드로 확장한 계기

처음 개발을 접했을 때는 C++을 통한 알고리즘 공부로 시작하게 되었습니다. 백준과 프로그래머스 문제들을 해결해 나가며 개발의 재미에 매료되었고, 더 나아가 게임 개발에 도전하게 되었습니다. 그 과정에서 시스템적으로 사고하는 법과 프로그래밍에 친숙해지는 계기가 되었습니다.

그러던 중, 코멘토에서 주관하는 웹 접근성, 웹 표준 직무 체험에 참여하게 되었습니다. 이전까지는 프론트엔드 개발을 단순히 눈에 보이는 UI를 구현하는 역할로 생각했지만, 이 경험을 통해 소외되는 사람이 없이 모든 사용자가 동등하게 웹에 접근할 수 있도록 보이지 않는 부분까지 세심하게 설계하고 구현해야 한다는 점을 깨닫게 되었습니다. 실제로 웹 접근성과 표준을 고려한 기능들을 직접 구현해보며, 사용자 경험을 보다 넓은 관점에서 고민하는 프론트엔드 직무가 아름답다고 느꼈고 결과적으로, 프론트엔드의 진정한 가치와 매력에 깊이 빠지게 되었습니다.

### 🏆 프로젝트 성과와 현재 집중 분야

이후 한컴 AI 아카데미 2기를 수료한 뒤, Don't Kill My Vibe 팀 프로젝트에서 프론트엔드 개발을 담당했습니다. VS Code Extension을 통해 개발 과정에서 AI를 활용한 바이브 코딩 실력을 실시간으로 점수화하고, 해당 결과를 웹에 저장 및 시각화하여 다양한 통계를 제공하는 서비스를 구현했으며, 그 결과 최우수 팀으로 선정되었습니다. 이 과정에서 단순히 AI를 사용하는 것을 넘어, 개발 과정에서 AI가 어떤 방식으로 가치를 만들어낼 수 있는지에 대해 깊이 고민하게 되었고 기술을 도구로 활용하는 관점뿐만 아니라 문제 해결의 새로운 접근 방식으로 바라보는 계기를 얻게 되었습니다.

현재는 React, Next.js, TypeScript 중심의 웹 개발에 집중하고 있습니다. 아직 배워가는 단계이지만 유지보수하기 쉽고 구조가 명확한 애플리케이션을 만드는 것을 목표로 다양한 프로젝트를 제작하며 성장하고 있습니다.
